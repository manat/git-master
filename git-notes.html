## Snapshots ![Normal VCS](http://git-scm.com/figures/18333fig0104-tn.png "Normal VCS") ![Git](http://git-scm.com/figures/18333fig0105-tn.png "Git") After all, git tracks contents and not filenames. When you add a file to your repo, the filename is in the tree object. The actual content of the file gets added as a blob object in your repo. If you add another file of the same content, git will not add another blob for it. See [The Git Object Model](http://schacon.github.io/gitbook/1_the_git_object_model.html) for more information. ## States ![Git States](http://graph1zzlle.github.io/images/posts/git-commit-workflow.png "Git States") ## Walkthrough * [Git Immersion](file:///C:/Users/dt77850/Documents/Git/tutorials/gitimmersion/git_tutorial/html/lab_01.html) * [Interactive Git](http://pcottle.github.io/learnGitBranching/?NODEMO) Starts Git basic with Git Immersion (1 - 25).
